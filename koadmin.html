Kompetencje Techniczne

Kompleksowe zarządzanie infrastrukturą z naciskiem na stabilność, wydajność i zgodność z regulacjami.

Administracja Hybrydowa

Zarządzanie środowiskami Linux i Windows Server. Kompleksowa obsługa wirtualizacji (Proxmox, Docker) oraz infrastruktury chmurowej (GCP, Terraform).

  • Linux / Windows Server
  • Proxmox Clusters
  • Konteneryzacja (Docker/ECS)

Bezpieczeństwo i NIS2

Wdrażanie polityk bezpieczeństwa zgodnych z dyrektywą NIS2. Konfiguracja SIEM/XDR oraz zarządzanie procedurami Disaster Recovery.

  • Backup & Disaster Recovery
  • Monitoring (Wazuh, SIEM)
  • VPN & Network Security

Sieci i Bazy Danych

Projektowanie i utrzymanie sieci LAN/WAN (Mikrotik, TCP/IP). Administracja i optymalizacja baz danych SQL w środowiskach produkcyjnych.

  • MS SQL, PostgreSQL, Firebird
  • Konfiguracja LAN/WAN
  • Systemy ERP

Automatyzacja (DevOps)

Tworzenie skryptów automatyzujących rutynowe zadania, raportowanie i przetwarzanie danych. Infrastructure as Code.

  • Python (Pandas), Bash
  • Ansible / Terraform
  • Integracje API / CI/CD
